## 🚀 To run this project on your local machine.

1. Download an extension or node.js
```sh
https://nodejs.org/en/download/
```
2. Go to the project directory.
```sh
cd <project folder>
```
3. Make a new terminal and run the command.
```sh
 node gauss
 ```
4. Press tab then hit enter.
